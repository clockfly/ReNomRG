Install ReNomRG
===============

ReNomRG requires following python modules.


Install from source.
~~~~~~~~~~~~~~~~~~~~

    If you install ReNomRG from source, Node.js is required.

    .. code-block:: shell

        git clone ~~
        cd ReNomRG/
        pip install -r requirements.txt
        pip install -e .
        python setup.py build

    .. note ::

        This requires ``node.js``.
